## 📌 Project Description

This project was developed as part of a Machine Learning coursework.

It includes two main tasks:

1. Customer Segmentation using clustering techniques
2. Credit Card Fraud Detection using classification models
3. AutoML exploration using TPOT

---

## 📊 Dataset 1: Customer Segmentation

The dataset contains information about mall customers including:

- Age
- Gender
- Annual Income
- Spending Score

### Methods used:
- Data exploration (info, describe)
- Data visualization (histograms, countplots)
- K-Means Clustering
- Agglomerative Clustering

### Goal:
To segment customers into different groups based on their behavior.

---

## 💳 Dataset 2: Fraud Detection

This dataset contains credit card transactions labeled as:

- Normal transactions
- Fraudulent transactions

### Methods used:
- Data preprocessing
- Logistic Regression
- Random Forest Classifier
- Model evaluation (Accuracy, Precision, Recall, F1-score)

### Challenge:
The dataset is highly imbalanced.

---

## 🤖 AutoML (TPOT)

TPOT was used to automatically search for the best machine learning pipeline.

However, due to computational limitations in Google Colab, full execution was not always successful.

The concept of AutoML was studied and compared with manual models.

---

## 📈 Results

- Clustering successfully identified customer groups
- Classification models achieved good performance in fraud detection
- Random Forest performed better than Logistic Regression
- AutoML concept was explored and analyzed
